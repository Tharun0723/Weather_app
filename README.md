# Weather App 🌦️

A Python CLI Weather Application that fetches real-time weather data using OpenWeatherMap API.

## Features
- Fetch weather by city name
- Displays temperature, humidity, and condition
- Uses REST API
- Error handling

## Technologies Used
- Python
- Requests library
- OpenWeatherMap API

## Setup Instructions

1. Install Python
2. Install required library:

pip install -r requirements.txt

3. Get free API key from OpenWeatherMap
4. Replace API key inside weather_app.py
5. Run:

python weather_app.py

## Project Purpose

This project demonstrates:
- API integration
- JSON parsing
- Real-time data fetching
- Error handling
